## 🚀 Overview

This project is an **Android Mobile Application** developed during my internship at PraxWare Technologies.

The application demonstrates core Android development concepts such as:

* User Authentication (Login & Registration)
* Navigation Components
* UI Design using XML
* Menu & Fragment Handling
* Basic API Concepts

Developed as part of a **15-day Android internship**, focusing on real-world mobile app development and UI/UX implementation.

---

## 🎯 Features

* 🔐 Login & Registration System
* 🏠 Home Dashboard
* 📊 Bottom Navigation Menu
* 📱 Multiple Activity Screens
* 📂 Menu-based Navigation
* 🔗 Basic API Concepts
* 🎨 Clean UI using XML Layouts

---

## 🛠️ Tech Stack

## ⚙️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/himanshu6297/Android-Mobile-Application.git
```

2. Open the project in **Android Studio**

3. Sync Gradle

4. Run the app on:

   * Android Emulator OR
   * Physical Device
